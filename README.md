# KennKeeper — Keep Your Friendships Alive 

    <p>By this project you can keep your friendsship alive. By this website you can find your friends and can see his/her details, status, contact info, contact status and many more. Also you can start a call, text and video call with your friend and can see the interactions with your friends on Timeline page and can see statictics on stat page.</p>


## Technologies Used: 

<p>• React.js</p>

<p>• React Router</p>

<p>• Tailwind CSS</p>

<p>• JavaScript (ES6+)</p>

<p>• React-Toastify</p>

<p>• Recharts</p>

<p>• Lucide Icons</p>

<p>• JSON (for product data)</p>


## Features:

<p>• By clicking the friends card it shows friends details with three functionable button. Those buttons are call, text and video.</p>

<p>• By clicking call, text and video button it shows an success toast with relavent text and stores the data to timeline page.</p>

<p>• On Timeline page it shows the recent activites with friends. it has a filter button with three filter item. By clicking those item it shows relavent interactions. In Stat page it shows statictics of call, text and videos.</p>
